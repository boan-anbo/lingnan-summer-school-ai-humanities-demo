# Material Analysis

Use after source collection, with the human reading step central.
Open the source on request and point the user to likely pages or passages.
Let the user inspect the page and make page-referenced notes in ordinary language.
When asked, turn the user's note and nearby source wording into an evidence record, not a final argument.
For PDFs, use page number, visible title/date, nearby wording, and any screenshot or copied passage the user provides.
Output: source, page/location, user note, quote/nearby text, possible reading, uncertainty/next check.
