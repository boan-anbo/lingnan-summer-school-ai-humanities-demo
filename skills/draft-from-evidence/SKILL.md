# Draft From Evidence

Use after evidence records exist; do not find sources or extract annotations here.
Draft the requested format from the evidence and the user's judgment.
Separate evidence, inference, and uncertainty; do not strengthen a claim beyond what the sources support.
For LaTeX, just write LaTeX; use BibLaTeX/Chicago if requested, or simple footnotes if not.
Cite close to each factual claim with source title/path/URL and page/location when available.
Do not cite AI output as historical evidence; cite it only as an object being evaluated.
If evidence is thin or conflicting, say so briefly in the draft.
Return only the draft plus a short note on missing evidence if needed.
