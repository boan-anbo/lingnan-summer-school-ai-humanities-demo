# Finding Source Candidates

Collect one best credible open-internet source candidate. Do not decide the final answer.

Prefer original or published materials, especially files an agent can inspect: PDFs, scans, records, repository files, books, articles, or reports.

Use search results, AI answers, and overview pages only as leads to citations, files, titles, authors, dates, and variant names.

Do not stop at a lead when it points to an underlying source. Follow the trail unless time or tool access blocks you.

If no original or published source is found quickly, return the best lead and state what is missing.

Save/download the single best source file when possible. Otherwise record URL/citation and access limits.

Return one compact record: query/path, title/owner, URL/citation, date/author, source type, relevant wording, local path, why best, problems, next check.
