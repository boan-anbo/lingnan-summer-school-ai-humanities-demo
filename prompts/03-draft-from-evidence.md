# Scene 3 Prompt: Draft From Evidence

Use the skill in `skills/draft-from-evidence/SKILL.md`.

Use the evidence record in:

`examples/lingnan-origin-evidence-record.md`

Task:

Write a short source-backed report paragraph in LaTeX.

Rules:

- Separate evidence, inference, and uncertainty.
- Do not strengthen the claim beyond what the source supports.
- Cite close to the factual claim.
- Mention that the later retrospective account is useful but not final.
- Return only the draft.
